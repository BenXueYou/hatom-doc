### 设置顶部状态栏模式

  使用示例：
  ```javascript
  hatom.statusBar.setStatusBarMode({
    iconMode: 'dark'
  },
  (res) => {
    console.log(res)
  })
  ```
  参数说明：
  
  | 参数   | 类型 | 必填 | 描述 |
  | -------- | ---- | ----- | -------- |
  | iconMode | string | 是 | 状态栏模式， dark模式， light模式 |

  &nbsp;

  返回说明：

  无

  &nbsp;
