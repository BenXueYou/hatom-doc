## 配置H5资源包
点击上传H5应用组件包，导入开发好的H5组件包，组件包需要使用 hatom2-cli 脚手架开发并打包为 zip 格式压缩包。
[H5应用组件包开发指南](../h5-developer/H5-developer.md)


## 配置组件
配置组件有两种方式，一是点击"添加组件"按钮去组件广场选择组件，二是在点"应用配置"按钮在模板配置页选择所需的模板快捷添加组件套装。
