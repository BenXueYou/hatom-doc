### 容器操作说明

#### 退出容器 单页面应用中，即为退出应用， 多页面则为退出该webview 容器

使用示例：
```
hatom.page.exit();
```
参数说明：

| 参数   | 类型 | 必填 | 描述     |
| -------- | ---- | ----- | -------- |
| callback | 函数 | 否 | 回调函数 |

&nbsp;

返回说明：

无

&nbsp;

#### 返回上一步

使用示例：
```
hatom.page.back();
```
参数说明：

| 参数   | 类型 | 必填 | 描述     |
| -------- | ---- | ----- | -------- |
| callback | 函数 | 否 | 回调函数 |

&nbsp;

返回说明：

无

&nbsp;
